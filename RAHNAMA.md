# راهنمای راه‌اندازی EsmFamil Local

## بخش اول: آماده‌سازی آیکون برنامه (تنها کار شما)
شما باید ۵ فایل عکس با نام دقیق `ic_launcher.png` بسازید و در پوشه‌های زیر قرار دهید:

- `frontend/android/app/src/main/res/mipmap-mdpi/`  →  ۴۸×۴۸ پیکسل
- `frontend/android/app/src/main/res/mipmap-hdpi/`  →  ۷۲×۷۲ پیکسل
- `frontend/android/app/src/main/res/mipmap-xhdpi/` →  ۹۶×۹۶ پیکسل
- `frontend/android/app/src/main/res/mipmap-xxhdpi/`→ ۱۴۴×۱۴۴ پیکسل
- `frontend/android/app/src/main/res/mipmap-xxxhdpi/`→ ۱۹۲×۱۹۲ پیکسل

از سایت‌هایی مثل easyappicon.com یا icon.kitchen برای تولید خودکار اندازه‌ها استفاده کن.

## بخش دوم: ساخت ریپازیتوری در GitHub
به github.com برو، یک ریپازیتوری Public جدید بساز.

## بخش سوم: آپلود
کل محتویات این پوشه را با Drag & Drop در گیت‌هاب آپلود کن. ساختار پوشه‌ها باید دقیقاً حفظ شود.

## بخش چهارم: ساخت خودکار
به تب Actions برو. ساخت بین ۸ تا ۱۵ دقیقه طول می‌کشد.

## بخش پنجم: دانلود APK
وقتی تیک سبز ظاهر شد، روی اکشن کلیک کن و از قسمت Artifacts فایل `EsmFamilLocal-debug-apk` را دانلود کن.
